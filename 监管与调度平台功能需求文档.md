# 监管与调度平台功能需求文档

# 一、整体布局框架

PC 端：监管平台面向政府主管部门（发改委、能源局等），核心职责是对辖区内充电设施运营、车网互动活动、电网安全、市场秩序进行宏观监控与合规管理，不直接操作场站设备，但拥有应急调度权限和数据全量接入权限。

页面整体结构：顶部固定信息栏 → 左侧固定功能导航栏 → 主内容展示区

# 二、顶部固定信息栏

## 左侧区域

平台 LOGO \+ 平台名称「监管与调度平台」

## 右侧区域（从左到右依次排列）

- **监管区域切换**：下拉选择监管层级（市 / 区 / 街道），默认显示当前登录用户管辖区域，切换后主内容区所有数据同步刷新。示例：「深圳市 ▼ 南山区 ▼」

- **登录用户信息**：显示「角色 \+ 姓名」（如：市级监管员 \| 张主任），点击下拉可修改密码、查看操作日志、退出登录

- **消息告警铃铛**：铃铛图标 \+ 未读红点（显示未处理告警数量），紧急事件（设备故障率异常、负荷超限、安全事故）弹窗 \+ 声音提醒，点击跳转「安全与应急管理 \- 安全告警监控」

- **手动刷新**：一键刷新主内容区所有动态数据，刷新时显示加载动画

- **退出登录**：点击弹出二次确认弹窗，确认后返回登录页

# 三、左侧固定功能导航栏

严格 6 个一级菜单，无二级菜单，一键直达；一级菜单（图标 \+ 文字）按监管工作流程排序（从宏观监控→具体业务监管→安全应急→系统管理），核心菜单如下：

|菜单名称|功能简述|
|---|---|
|数据总览|辖区充电运营核心指标看板（登录默认进入）|
|设施监管|充电桩 / 站基础信息、运行状态、接入合规性|
|车网互动与调度|V2G、有序充电、需求响应、智能功率分配、调度模型结果|
|运营与合规管理|运营商资质、服务评级、违规管理、补贴审核、政策发布|
|安全与应急管理|安全告警、应急调度指令、事故追溯|
|系统管理|用户权限、数据接入、告警规则、运营分析、操作日志|

# 四、主内容展示区（各模块完整功能）

## （一）数据总览模块（登录默认页面，运营核心看板）

功能描述：宏观展示辖区内充电设施运营与车网互动的核心指标，一眼看清整体运行态势。所有卡片数据支持点击钻取到对应模块详情。

整体布局：顶部核心指标卡片区（占页面 1/4）→ 中部趋势图表区（占页面 1/2）→ 底部告警提醒区（占页面 1/4）

### 1\. 核心指标卡片区（顶部通栏，9 个数字卡片，一行 4 个）

|卡片名称|展示内容|点击跳转|
|---|---|---|
|充电设施接入|充电站总数 / 充电桩总数 / V2G 桩数及占比|设施监管 \- 设施台账|
|今日充电|今日充电总量（万 kWh）/ 环比昨日增减|系统管理 \- 充电量分析|
|今日 V2G 放电|今日 V2G 放电总量（万 kWh）/ 参与车辆数|车网互动与调度 \- V2G 活动监管|
|需求响应|累计削峰响应电量（MWh）/ 累计填谷电量（MWh）|车网互动与调度 \- 需求响应监管|
|运营商接入|运营商数量 / 本月新增|运营与合规管理 \- 运营商管理|
|设备在线率|当前在线率 / 环比昨日|设施监管 \- 运行状态监控|
|故障告警|当前未处理告警数 / 严重告警数（红色高亮）|安全与应急管理 \- 安全告警监控|
|补贴发放|本年度已发放补贴总额（万元）|运营与合规管理 \- 补贴管理|
|调度执行|调度执行率 / 超功率站点数|车网互动与调度 \- 智能调度|

### 2\. 趋势图表区（中部，左右分栏）

- **左侧 \- 充放电趋势图**：


    - 图表类型：双轴折线图（充电量 \+ 放电量）

    - 时间维度：近 7 天 / 近 30 天（Tab 切换），支持按日 / 周 / 月聚合

    - X 轴：日期，Y 轴左：充电量（万 kWh），Y 轴右：放电量（万 kWh）

    - 交互：鼠标悬浮显示当日具体数值，点击图表任意点跳转「系统管理 \- 运营分析」查看详情

- **右侧 \- 需求响应效果图**：
        

    - 图表类型：堆叠柱状图

    - 展示内容：近 7 天 / 近 30 天各运营商参与响应次数与响应电量

    - X 轴：日期，Y 轴：响应电量（MWh），柱内按运营商颜色区分

    - 底部展示「响应完成率」折线（目标 vs 实际）

    - 交互：点击柱状图跳转「车网互动与调度 \- 需求响应监管」

### 3\. 告警实时提醒区（底部，红色高亮列表）

按紧急程度排序（严重告警→警告→提示），展示：告警类型、发生区域、所属运营商、发生时间、告警内容

每条告警右侧「查看详情」按钮，点击跳转「安全与应急管理 \- 安全告警监控」

- 严重告警自动弹窗 \+ 声音提醒，持续 10 秒或用户手动关闭

- 支持按告警级别筛选（全部 / 严重 / 警告 / 提示）

## （二）充电设施监管模块

功能描述：对辖区内所有充电设施进行基础信息管理、运行状态监控、接入合规性审查。

整体布局：顶部 Tab 栏（设施台账 / 接入审核 / 运行状态监控）→ 主体表格区 → 底部分页

### 1\. 充电设施台账（Tab1）

功能：展示辖区内所有充电站 / 桩的基础信息，支持查询、筛选、导出。

#### 顶部筛选栏：

区域筛选（市 / 区 / 街道，级联下拉）、运营商筛选（下拉，支持多选）、接入状态筛选（全部 / 已接入 / 断连 / 待审核）、设施类型筛选（全部 / 直流快充 / 交流慢充 / V2G）、关键词搜索（站点名称 / 地址 / 设备编号）、「重置」按钮清空所有筛选条件、「导出」按钮导出当前筛选结果（Excel）

#### 主体表格（字段说明）：

|字段名称|说明|示例|
|---|---|---|
|站点名称|充电站名称，点击跳转站点详情页|南山科技园充电站|
|所属运营商|运营商名称，点击跳转运营商详情|南网电动|
|区域|行政区划|深圳市南山区|
|地址|详细地址|科技路 10 号|
|充电桩总数|总桩数（直流 / 交流 / V2G）|20（8/6/6）|
|V2G 桩数|具备 V2G 功能的桩数量|6|
|投运时间|站点投入运营日期|2024\-03\-15|
|接入状态|已接入 / 断连 / 待审核，断连标红|已接入|
|最后上报时间|最后一次数据上报时间|2025\-01\-15 14:30:00|

#### 站点详情页（钻取页面）：

点击站点名称进入，展示该站点完整信息：

- 基本信息：站点名称、地址、运营商、投运时间、联系方式

- 设备清单：表格展示所有充电桩（设备编号、类型、功率、状态、最后通讯时间）

- 历史运行数据：近 30 天充电量趋势图、在线率趋势图

- 告警记录：该站点近 90 天所有告警列表

- 返回按钮回到设施台账

### 2\. 接入审核（Tab2）

功能：审核充电站 / 桩的接入申请，通过后方可纳入监管平台监控。

#### 顶部筛选栏：

审核状态筛选（全部 / 待审核 / 已通过 / 已驳回）、申请时间筛选（最近 7 天 / 本月 / 自定义）、关键词搜索（站点名称 / 运营商名称）

#### 主体表格：

|字段名称|说明|
|---|---|
|申请编号|系统自动生成，唯一标识|
|运营商名称|申请方运营商|
|站点名称|待接入站点名称|
|地址|站点地址|
|设备信息|桩数量 / 类型 / 功率汇总|
|申请时间|提交申请的时间|
|审核状态|待审核 / 已通过 / 已驳回|
|操作|「查看详情」「审核」按钮|

#### 审核详情页：

展示完整的申请资料：

- 站点信息：名称、地址、坐标、联系人、联系电话

- 设备清单：表格展示每台设备的型号、功率、类型、序列号

- 资质文件：营业执照、电力接入许可、安全承诺书（支持在线预览 / 下载）

- 现场照片：站点环境照片、设备照片

审核操作区：

- 审核意见输入框（必填，驳回时必填驳回理由）

- 「通过」按钮：点击后状态变为「已通过」，该站点纳入监管

- 「驳回」按钮：点击后状态变为「已驳回」，系统自动通知运营商

- 二次确认弹窗：「确认通过该站点接入申请？」

### 3\. 运行状态监控（Tab3）

功能：实时监控辖区内设备的运行状态，发现异常设备自动告警。

#### 顶部筛选栏：

区域筛选、运营商筛选、状态筛选（全部 / 正常 / 故障 / 离线）、故障类型筛选（通讯中断 / 充电枪故障 / 功率异常 / 过温 / 其他）

#### 主体表格：

|字段名称|说明|
|---|---|
|设备编号|充电桩唯一编号，点击查看设备详情|
|所属站点|站点名称，点击跳转站点详情|
|所属运营商|运营商名称|
|设备类型|直流快充 / 交流慢充 / V2G|
|运行状态|正常 / 故障 / 离线（故障 / 离线标红）|
|故障类型|如「通讯中断」，正常状态显示「\-」|
|故障时间|首次出现故障的时间|
|处理状态|待处理 / 处理中 / 已闭环（来自运营商上报）|
|操作|「下发整改通知」按钮（生成 PDF 通知，可下载）|

#### 设备详情页（钻取页面）：

点击设备编号进入，展示：

- 设备档案：设备编号、型号、厂家、投运时间、维保到期时间

- 实时参数：当前电压 / 电流 / 功率（如有上报）

- 近 24 小时运行曲线：电压 / 电流 / 功率趋势图

- 故障记录：历史故障列表及处理记录

- 整改通知记录：已下发的整改通知列表

## （三）车网互动与调度模块

功能描述：对 V2G 活动、有序充电执行、需求响应参与情况、智能调度功率分配结果进行全过程监管与可视化展示，基于电网约束、负荷预测、场站能力自动计算最优充放电功率，实现全网资源统一调控与执行跟踪。

整体布局：顶部 Tab 栏（V2G 活动监管 / 有序充电监管 / 需求响应监管 / 智能调度 / 车网互动效果统计）

### 1\. V2G 活动监管

功能：展示辖区内所有 V2G 活动，监控活动执行情况。

#### 顶部筛选栏：

时间范围筛选（近 7 天 / 本月 / 自定义）、区域筛选、活动状态筛选（全部 / 进行中 / 已结束 / 已取消）

#### 主体表格：

|字段名称|说明|
|---|---|
|活动名称|V2G 活动名称，点击查看活动详情|
|活动时间|起止时间（如：2025\-01\-15 19:00\-22:00）|
|活动区域|行政区划|
|参与站点数|实际参与的充电站数量|
|参与车辆数|实际参与的车辆数量|
|放电总量|活动期间累计放电量（kWh）|
|平均放电收益|用户平均每度电收益（元 /kWh）|
|活动状态|进行中 / 已结束 / 已取消|
|操作|「查看详情」「导出报告」|

#### 活动详情页（钻取页面）：

- 基本信息：活动名称、时间、区域、组织方（电网 / 运营商）

- 执行情况：放电功率曲线（实际 vs 目标），折线图展示；参与运营商排名（按放电量排序）；放电量分布图（按站点 / 按区域）

- 异常监控：异常事件列表（放电功率异常波动、设备离线等）；异常处理记录

- 效果评估：削峰贡献评估、用户收益汇总

- 支持导出活动报告（PDF）

### 2\. 有序充电监管

功能：监控接入有序充电策略的场站执行情况。

#### 顶部筛选栏：

区域筛选、运营商筛选、策略状态筛选（全部 / 执行中 / 已暂停 / 已结束）

#### 主体表格：

|字段名称|说明|
|---|---|
|场站名称|充电站名称，点击跳转场站详情|
|所属运营商|运营商名称|
|策略类型|台区容量约束 / 光伏消纳优先 / 电价优化|
|策略执行率|实际执行时段占比（百分比）|
|负荷平移电量|峰时转移至谷时的充电量（kWh）|
|谷时充电占比提升率|策略实施前后对比|
|执行状态|执行中 / 已暂停 / 已结束|
|操作|「查看执行详情」「标记异常」|

#### 异常监控：

底部展示「异常场站列表」：未按策略执行的场站（超容量充电 / 峰时充电比例异常），每条异常记录可「下发整改通知」

### 3\. 需求响应监管

功能：监管需求响应活动的组织与执行情况。

#### 顶部筛选栏：

响应类型筛选（全部 / 削峰响应 / 填谷响应）、时间范围筛选、响应状态筛选（全部 / 进行中 / 已结束）

#### 主体表格：

|字段名称|说明|
|---|---|
|响应活动名称|活动名称，点击查看详情|
|响应时间|起止时间|
|响应类型|削峰响应 / 填谷响应|
|响应目标|目标响应电量（MWh）|
|实际响应量|实际完成响应电量（MWh）|
|完成率|实际 / 目标 ×100%（低于 80% 标红）|
|参与运营商数|参与活动的运营商数量|
|操作|「查看详情」「导出执行报告」|

#### 活动详情页：

- 响应曲线对比：目标功率曲线 vs 实际功率曲线（双线折线图）

- 运营商完成率排名：表格展示各运营商响应电量、完成率、排名

- 不合格标记：完成率低于 80% 的运营商自动标记「响应不合格」，可作为补贴扣减依据

- 支持导出活动报告（含各运营商完成明细）

### 4\. 智能调度

功能描述：展示调度模型计算结果，全区调度概览 \+ 各站点充放电功率分配明细在同一页面展示，支持查看、筛选、人工调整与指令下发，实现 “看总、看细、可调、可下发” 一站式调度。

整体布局：顶部核心指标卡片 → 中部可视化图表 → 底部站点功率分配明细表格

#### （1）顶部核心指标卡片（调度总览）

- 调度总充电功率：模型计算计划充电总功率（kW）

- 调度总放电功率：模型计算计划放电总功率（kW）

- 实际执行功率：当前实时充电 / 放电总功率（kW）

- 调度执行率：实际 / 计划 ×100%（低于 80% 标红）

- 可调度站点数：正常在线可参与调度站点数量

- 超功率站点数：超出分配功率上限站点数（红色高亮）

- 电网约束状态：台区负荷正常 / 越限提示

- 调度模式：自动调度 / 人工干预

#### （2）中部调度可视化（总览趋势）

- 区域功率分配柱状图：各区域计划充电、放电功率对比

- 全网功率趋势图：目标功率曲线 vs 实际功率曲线

- 场站负荷热力图：颜色越深分配功率越高，红 = 超上限、黄 = 接近上限、绿 = 正常

#### （3）底部站点功率分配明细（核心・不重复）

##### 顶部筛选栏

调度任务筛选、区域筛选（市 / 区 / 街道）、运营商筛选、调度类型：充电 / 放电 / 双向调度、执行状态：全部 / 已执行 / 待执行 / 异常、关键词搜索、重置、导出 Excel

|字段名称|说明|
|---|---|
|站点名称|点击进入调度详情|
|所属运营商|运营商名称|
|区域|市 / 区 / 街道|
|最大允许充电功率|模型计算充电上限（kW）|
|最大允许放电功率|模型计算放电上限（kW）|
|调度总功率上限|充放电综合约束（kW）|
|当前实际充电功率|实时上报值（kW）|
|当前实际放电功率|实时上报值（kW）|
