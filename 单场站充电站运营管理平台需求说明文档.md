# 五、主内容展示区模块详情

## 5\.1 场站总览（登录默认页）

极简可视化核心数据看板，点击数据可钻取对应模块

- 核心设备状态区：总充电桩数、在线 / 故障 / 离线数、V2G / 储能 / 光伏设备运行状态

## 5\.2 设备管理（融合负荷状态展示）

设备全生命周期管理

- 设备档案：设备编号、类型、位置、投运 / 维保时间、厂家信息，支持编辑、维保记录、Excel 导出

- 实时监控（10 秒刷新）：按区域展示设备状态、实时参数、关联订单，可查看 24 小时运行曲线

- 远程控制：单设备启停充电、调节功率、V2G 放电、储能 / 光伏启停，操作需二次确认

- 故障处理：故障列表、处理标记、照片上传、闭环管理，维保到期提前 7 天提醒

- OTA 升级：设备版本查看、在线升级、失败回滚，升级记录可查

- 场站负荷状态：实时总负荷、变压器额定负荷、负荷率，24 小时趋势图，超 90% 告警

## 5\.3 充电运营

充电全流程管理，无跨场站功能

- 充电订单管理：订单列表、多维度筛选、详情查看、异常退费、发票申请、数据导出

- 实时充电监控：当前充电信息展示，支持主动结束充电

- 充电预约管理：预约列表、确认 / 取消 / 自动作废，成功弹窗提醒

- 现场用户管理：用户信息、消费记录查询，会员等级调整

- 订单对账管理：按日对账、标记状态，一键生成日 / 月对账报表

- 费率配置：峰 / 平 / 谷单价、会员折扣、V2G 放电单价，保留调整记录可追溯

单场站数据统计与辅助决策
